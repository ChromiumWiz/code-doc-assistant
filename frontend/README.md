# Code Documentation Assistant Frontend

Minimal Next.js 14 (App Router) UI for the Code Documentation Assistant backend.

## Setup
1. Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_API_BASE_URL`.
2. Install deps and run:

```bash
npm install
npm run dev
```

## Pages
- `/` Create Repo
- `/index` Index Repo
- `/chat` Chat
